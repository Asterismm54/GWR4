# GWR4
Tools for GWR4

Software to perform Geographically Weighted Regression

GWR4 is application software for calibrating geographically weighted regression (GWR) models, which can be used to explore geospatial relationships between dependent variables and independent variables. This software allows fit to semiparametric GWR models. A GWR model is considered a type of regression model with geographically varying parameters.
